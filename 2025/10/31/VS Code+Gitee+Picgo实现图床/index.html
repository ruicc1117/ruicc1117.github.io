VS Code+Gitee+Picgo实现图床
Keep Team Lv1
  2025-10-31 00:55:33   2025-01-12 16:32:02

在VSCode中结合Gitee图床和PicGo插件，解决Markdown文档插入图片的问题。

步骤一、在VSCode中安装Picgo插件

步骤二、在系统中安装Picgo软件

  1. 进入PicGo官网：https://molunerfinn.com/PicGo/ 。
  2. 下载最新版本.exe文件。
  3. 安装完成后，打开PicGo，点击插件设置，搜索gitee，安装gitee-uploader1.1.2。
  4. 点击安装后发生错误, 主要是由于PicGo软件是基于node.js, 用electron-vue开发的客户端, 这里我要去node.js官网下载安装一下。
  5. 进入Node.js官网：https://nodejs.org/en/download 下载适用于win系统的.msi文件。根据引导进行安装。
  6. 重启PicGo，安装gitee-uploader1.1.2插件。

步骤三、创建Gitee仓库用于上传图片

设置仓库名和路径；设置仓库为开源；设置模板Read me文件。

步骤四、在PicGo软件中配置gitee

  • repo：用户名/仓库名。直接复制浏览器地址栏中的url。
  • branch：master。
  • token：填入gitee中的私人令牌。
  • path：gitee仓库文件夹名称，将图片上传到仓库中的文件夹。

如何获取token？

  1. 在gitee中点击右上角的设置。
  2. 点击私人令牌，然后生成新令牌。
  3. 选择相应的权限提交。
  4. 将生成的私人令牌填入Picgo的token输入框。然后设置为默认图床后确定。

结束语

此时图床设置完成，可用快捷键将剪切板中的图片上传至图床，将返回一个图片外链。
