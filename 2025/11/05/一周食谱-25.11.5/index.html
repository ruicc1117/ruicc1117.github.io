一周食谱-25.11.5

花生
2025-11-05
生活

由于自己对自己实在照顾不周，饮食上大多数靠外卖和方便面过活，于是在今天利用5分钟时间找了一份一周食谱，每天严格（不是非常严格）地按照食谱做饭用餐。

🌟 周一

早餐：全麦面包1片 + 水煮蛋1个 + 牛奶200ml + 蓝莓50g
午餐：糙米饭1碗（100g生米）+ 清蒸鲈鱼150g + 蒜蓉西兰花200g + 番茄豆腐汤
加餐：无糖酸奶100g + 杏仁10颗
晚餐：杂粮粥（燕麦+小米）+ 香煎鸡胸肉100g + 凉拌菠菜（芝麻油+醋）

🌟 周二

早餐：燕麦片30g（牛奶冲泡）+ 香蕉1根 + 核桃2颗
午餐：荞麦面（80g干面）+ 虾仁炒芦笋（虾仁80g+芦笋150g） + 紫菜蛋花汤
加餐：苹果1个 + 煮毛豆50g
晚餐：蒸红薯1个（中等大小）+ 西芹炒牛肉（牛肉100g+西芹200g） + 海带冬瓜汤

🌟 周三

早餐：蔬菜鸡蛋饼（全麦粉30g+鸡蛋1个+胡萝卜丝+菠菜碎） + 豆浆200ml
午餐：三色藜麦饭（生米80g）+ 烤三文鱼120g + 白灼秋葵150g
加餐：奇异果1个 + 无糖黑巧克力1小块（10g）
晚餐：玉米1根 + 香菇炖鸡（去皮鸡腿肉150g+香菇5朵） + 凉拌黄瓜

🌟 周四

早餐：希腊酸奶150g + 奇亚籽5g + 草莓5颗 + 全麦脆片20g
午餐：杂粮饭（糙米+黑米）+ 豆腐抱蛋（嫩豆腐150g+鸡蛋1个） + 清炒空心菜200g
加餐：橙子1个 + 煮鸡蛋1个
晚餐：南瓜小米粥 + 清蒸鳕鱼100g + 芦笋炒口蘑（200g）

🌟 周五

早餐：紫薯1个（小）+ 牛奶煮燕麦30g + 水煮西兰花50g
午餐：意面（生重60g）+ 番茄牛肉酱（瘦牛肉末80g+番茄200g） + 橄榄油拌羽衣甘蓝沙拉
加餐：火龙果半颗（约100g） + 腰果8颗
晚餐：蒸芋头100g + 白灼虾10只 + 凉拌木耳黄瓜（200g）

🌟 周六

早餐：全麦三明治（全麦面包2片+煎蛋+生菜+番茄片） + 无糖拿铁
午餐：糙米饭1碗 + 麻婆豆腐（少油版，豆腐150g） + 清炒芥蓝200g + 萝卜排骨汤（排骨去脂）
加餐：梨1个 + 无糖酸奶100g
晚餐：山药红枣粥（山药100g）+ 香煎龙利鱼150g + 蒜蓉菜心200g

🌟 周日

早餐：蔬菜鸡肉粥（鸡胸肉50g+大米30g+菠菜+香菇） + 水煮蛋1个
午餐：土豆泥（土豆150g）+ 烤鸡腿（去皮1只） + 凉拌苦菊彩椒（200g）
加餐：葡萄15颗 + 原味坚果20g
晚餐：红豆薏米粥 + 清炒莴笋丝200g + 酱牛肉80g
